# 1.如何创建分支

    git branch 分支的名字  // 仅仅是创建

# 2.如何切换分支/检出分支

    git checkout 分支的名字  // 仅仅是切换、检出

    git checkout -b 分支的名字  // 创建后，切换到新分支
